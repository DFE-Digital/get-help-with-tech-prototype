We recommend checking with independent settings as soon as possible to find out which device type and IT settings would be most suitable for each pupil.

Depending on availability, you can choose from:

Standard Windows laptops

You’ll be responsible for arranging and maintaining safeguarding measures, like content filtering.

DfE Restricted Windows laptops

These have temporary content filtering and mobile device management software installed, which you’ll need to replace when they expire. You’ll need to reset these devices before you can apply new settings, or download some types of software.

Google Chromebooks

You’ll need to provide us with the relevant school’s email domain for G Suite for Education, and a recovery email address.
